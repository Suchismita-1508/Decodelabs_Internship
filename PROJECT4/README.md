# 🛡️ System Vulnerability Checklist and Security Risk Assessment

## 📌 Project Overview

The **System Vulnerability Checklist and Security Risk Assessment** project is a Python-based cybersecurity application developed as part of the **DecodeLabs Cyber Security Internship (Batch 2026)**. The application performs a basic security audit by evaluating common security practices and identifying potential vulnerabilities in a system.

It analyzes user responses related to password security, operating system updates, antivirus protection, firewall status, multi-factor authentication (MFA), VPN usage, and USB security to calculate an overall security risk level and provide personalized recommendations.

---

## 🎯 Objective

The objective of this project is to help users identify common security vulnerabilities and improve their cybersecurity posture through a simple security assessment checklist.

The application:

- Performs a system security checklist
- Calculates a security risk score
- Identifies weak security practices
- Provides recommendations to improve security

---

## ✨ Features

- ✅ Checks password strength
- ✅ Verifies operating system updates
- ✅ Checks antivirus protection status
- ✅ Verifies firewall status
- ✅ Checks Multi-Factor Authentication (MFA)
- ✅ Verifies VPN usage on public Wi-Fi
- ✅ Checks unknown USB device usage
- ✅ Calculates an overall security risk score
- ✅ Classifies the system as Low, Medium, or High Risk
- ✅ Generates personalized security recommendations

## 📌 How It Works

1. The user answers a series of security-related questions.
2. The program evaluates each response.
3. Risk points are assigned for insecure practices.
4. An overall risk score is calculated.
5. The system classifies the security level.
6. Personalized recommendations are displayed to improve system security.
