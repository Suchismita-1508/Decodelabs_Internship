# 🔐 Basic Encryption & Decryption (Caesar Cipher)

A simple yet effective Basic Encryption & Decryption system developed in Python using the Caesar Cipher algorithm. This project demonstrates the fundamentals of cryptography by allowing users to encrypt plain text into cipher text and decrypt it back using a shift key. It serves as an excellent introduction to encryption techniques and secure data communication.

## ✨ Features

- Encrypts plain text using the Caesar Cipher algorithm
- Decrypts encrypted text back to its original form
- Supports user-defined shift key
- Preserves spaces and special characters
- Simple command-line interface
- Beginner-friendly Python implementation

## 🛠️ Technologies Used

- Python 3
- Built-in Python Functions (`ord()`, `chr()`)
- Modular Arithmetic

## 📌 How It Works

1. The user enters a message.
2. A shift key is provided.
3. The program encrypts the message by shifting each alphabet character.
4. The encrypted text is displayed.
5. The program decrypts the cipher text back to the original message for verification.
