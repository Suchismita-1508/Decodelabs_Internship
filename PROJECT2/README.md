## 🔐 Basic Encryption & Decryption (Caesar Cipher)

## 📌 Project Overview

The **Basic Encryption & Decryption** project is a Python-based cybersecurity application developed as part of the **DecodeLabs Cyber Security Internship (Batch 2026)**. It demonstrates the fundamentals of cryptography using the **Caesar Cipher** algorithm by allowing users to encrypt plaintext into ciphertext and decrypt it back using a user-defined shift key.

This project introduces the core concepts of data confidentiality, encryption logic, and secure communication.

---

## 🎯 Objective

The objective of this project is to implement a simple encryption and decryption system using the Caesar Cipher algorithm.

The application:

- Encrypts user input using a shift key
- Decrypts the encrypted message back to its original form
- Demonstrates the basic principles of cryptography and data protection

---

## ✨ Features

- ✅ Encrypts plaintext using the Caesar Cipher algorithm
- ✅ Decrypts ciphertext back to the original message
- ✅ Supports user-defined shift key
- ✅ Preserves spaces and special characters
- ✅ Simple command-line interface
- ✅ Beginner-friendly Python implementation

---

## 🛠️ Technologies Used

- Python 3
- Built-in Python Functions (`ord()`, `chr()`)

## 📌 How It Works

1. The user enters a message.
2. A shift key is provided.
3. Each alphabet character is shifted by the specified key.
4. The encrypted message is displayed.
5. The encrypted text is decrypted using the same key.
6. The original plaintext is restored.
