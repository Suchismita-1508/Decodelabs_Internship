# 📂 Projects

## 🔐 Project 1 – Password Strength Checker and Secure Password Validation Using Python

A Python application that evaluates password strength by checking:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

**Skills Learned**

- Password Security
- String Manipulation
- Regular Expressions
- Security Validation
- Authentication Basics

---

## 🔑 Project 2 – Basic Encryption and Decryption Using the Caesar Cipher Algorithm

A Python implementation of the Caesar Cipher that encrypts and decrypts user messages using a customizable shift key.

**Skills Learned**

- Cryptography
- Caesar Cipher
- ASCII Manipulation
- Modular Arithmetic
- Data Confidentiality

---

## 🎣 Project 3 – Phishing Awareness Analysis and Cyber Threat Detection Using Python

A phishing detection tool that analyzes emails and messages for suspicious links, keywords, urgency indicators, and other phishing characteristics.

**Skills Learned**

- Phishing Detection
- Threat Analysis
- Social Engineering Awareness
- Pattern Matching
- Risk Assessment

---

## 🛡️ Project 4 – System Vulnerability Checklist and Security Risk Assessment Using Python

A security assessment tool that evaluates common security practices and provides recommendations based on the calculated risk score.

**Skills Learned**

- Security Auditing
- Vulnerability Assessment
- Risk Analysis
- Cybersecurity Best Practices
- Security Compliance
